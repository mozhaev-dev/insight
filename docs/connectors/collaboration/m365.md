## Overview

**API**: Microsoft Graph API v1.0 — Report endpoints (`/reports/get*ActivityUserDetail`)

**Category**: Communication

**Identity**: `userPrincipalName` (UPN, email address) — resolved to canonical `person_id` via Identity Manager

**Field naming**: camelCase — original Graph API field names, no renaming at Bronze level.

**Why six tables**: The Graph API exposes Email, Teams, OneDrive, SharePoint, and Copilot activity as separate endpoints with different report periods and field sets. Each is stored as its own Bronze table sharing the same identity key (`userPrincipalName + reportRefreshDate`). The collection run log is a seventh monitoring-only table.

> **Critical: Data Retention Window**
> M365 Graph API returns only the **last 7–30 days** of activity. Data cannot be re-fetched once the window passes — loss is permanent. The collector must run at minimum every 7 days to avoid data gaps.

## Identity Resolution

`userPrincipalName` (UPN, email address) is the primary identity key across all M365 Bronze tables.

The Identity Manager resolves `userPrincipalName` → canonical `person_id` in Silver step 2. The M365 UPN is typically the corporate email address, making it a reliable cross-system identity anchor.

`ms365_teams_activity` also contains `userId` (Microsoft internal user ID) — this is not used for cross-system resolution; `userPrincipalName` takes precedence.

## Open Questions

### OQ-M365-1: Copilot endpoint availability and schema

The `getMicrosoft365CopilotUsageUserDetail` endpoint requires a Microsoft 365 Copilot license on the tenant. Field names in `ms365_copilot_usage` are proposed based on Graph API patterns — the live endpoint must be queried to confirm the exact response shape before implementation.

### OQ-M365-2: OneDrive and SharePoint Silver mapping

`ms365_onedrive_activity` and `ms365_sharepoint_activity` are collected but not yet unified. If a `class_file_storage` or similar Silver stream is defined, both tables feed it. Decision pending on whether file-storage metrics are in scope for Silver.
